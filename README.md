# agy-termux-va39

Run Google Antigravity CLI on Android Termux using VA39 patching and a Termux wrapper.

## Install

```bash
curl -fsSL https://raw.githubusercontent.com/why382/agy-termux-va39/main/install.sh | bash
```

Then run:

```bash
source ~/.bashrc
agy --version
agy
```

## Warning

Do not upload or redistribute agy or agy.va39. This installer downloads the official Antigravity CLI and patches it locally.
